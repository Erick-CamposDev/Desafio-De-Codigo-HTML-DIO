Oque é HTML?

Voltar

Hyper Markup Text Language mais conhecida como HTML é uma linguagem de marcação muito utilizada em desenvolvimento web

o HTML estrutura as páginas web utilizando tags que podem adicionar textos, imagens, vídeos etc.

O HTML é um dos 3 pilares do desenvolvimento web junto de CSS e JavaScript, e mesmo que não seja uma linguagem de programação

é umas das principais e mais importantes ferramentas que se utiliza nessa área da tecnologia!

Oque são tags?

Voltar

Basicamente, as tags são elementos que são utilizados para marcar e estruturar as páginas web.

Elas dizem ao navegador como o conteúdo vai ser organizado e exibido!

Tags de título

Voltar

As tags de titulos são tags que podem adicionar tanto títulos quanto subtítulos a página.

Essas tags vai de h1 até h6 onde o h1 é o maior e o h6 é o menor

Elas seguem uma hierarquia que ajudam a estruturar e ajudar na semântica do código HTML, ajudando os mecanismos de pesquisa do google

e leitores de tela a entenderem o contéudo da página!

Tags de parágrafos e quebra de linha

Voltar

Tag p

A tag p é utilizada para criar parágrafos de texto no seu código HTML.

O navegador adiciona espaços verticais antes e depois para separar bem os parágrafos.

Tag br

A tag br é utilizada para quebrar linhas dentro de parágrafos, ou seja, ela força o texto a continuar na linha de baixo.

A tag br não precisa de uma tag de fechamento já que por sua vez ela é uma tag vazia.

Negrito, Italico, Sublinhado

Voltar

Tags b e strong

As tags b e strong dão um destaque no texto, ou seja, dão uma coloração mais destacada como por exemplo: Negrito

Qual a diferença das tags b e strong?

A diferença se refere como o código e lido semânticamente, strong é mais recomendado pois o bot vai entender com mais clareza diferente do b

Tag i

A tag i serve para deixar as letras inclinadas para o lado para destacar a palavra como por exemplo: Itálico

Tag u

A tag u deixa um texto sublinhado ou seja coloca um risco embaixo da palavra para enfatizar uma palavra como por exemplo: Sublinhado

Links e Âncoras

Voltar

Tag a

A tag a serve para fazer links clicáveis que leva o usuário para outros sites, arquivos email etc.

Usa-se o atributo href para definir o destino do link.

Formulários

Voltar

A tag form é responsável por criar formulários que permitem o usuário a enviar dados para o servidor como e-mail, senha, nome etc.

Atributos

Seus atributos são:

  • Action

    Esse atributo irá definir aonde os dados serão enviados.

  • Method

    Define como os dados serão enviados seja GET(Dados na URL) ou POST(Dados ocultos).

Listas Ordenadas e não ordenadas

Voltar

Tag ol lista ordenada

A tag ol é responsável por criar uma lista numerada de 1 ao número que você quiser!

Geralmente é mais utilizada em receitas, instruções, rankings etc.

Tag ul lista não ordenada

A tag ul tem uma função bastante semelhante a tag ol, porém as listas não são numeradas!

No lugar de números marcadores são colocados como forma de colocar em tópicos.

Geralmente usada em listas de compras, características etc.

Tags de divisão

Voltar

Tag Div

É uma tag comum e genérica.

A div é muito utilizada para estruturar o código HTML e deixa-lo organizado.

Entretanto, não possui nenhum valor semântico.

Outras tags com valores semânticos

  • Header

    Tag para o Cabeçalho da página ou seção

  • Main

    Tag para o conteúdo principal do site

  • Footer

    Tag para o rodapé da página

  • Article

    Tag para conteúdo independente

  • Section

    Tag para conteúdo relacionado

Imagens

Voltar

A tag img é uma tag que é capaz de inserir imagens no seu site!

Ela é uma tag vazia, ou seja, não tem uma tag de fechamento.

Seus atributos são: src que é o local onde coloca a fonte da imagem, width e height para definir largura e altura e alt para texto de acessibilidade.

Tabelas

Voltar

A tag table é responsável por criar tabelas para seu site!

Tags utilizadas para a tabela

  • tr

    Tag que define a linha de uma tabela

  • th

    Tag que define uma célula do cabeçalho

  • td

    Tag que define uma célula de dado

  • Thead,Tbody,Tfoot

    Seções opcionais para uma organização melhor de tabela
